# v1.0.0

*Released: December 2024*

The initial release of Docs Starter Kit - an open-source documentation platform built with Laravel, Vue.js, and TypeScript.

## Features

### Dual Content Mode

Choose how you want to manage your documentation:

- **CMS Mode**: Full-featured admin panel with visual editor
- **Git Mode**: Sync documentation from GitHub repositories

### Rich Text Editor

TipTap-powered WYSIWYG editor with:

- Text formatting (bold, italic, headings)
- Code blocks with syntax highlighting
- Tables and lists
- Image embedding
- Link management

### Page Management

Hierarchical documentation structure:

- **Navigation tabs**: Top-level categories
- **Groups**: Sidebar sections
- **Documents**: Content pages

Features include:
- Drag-and-drop reordering
- Page duplication
- Version history
- Draft/publish workflow

### Git Synchronization

For developer documentation:

- Sync from GitHub (public and private repos)
- Webhook support for instant updates
- Automatic scheduled sync
- "Edit on GitHub" links
- Rollback to previous syncs

### Customization

Extensive theming options:

- Color configuration (primary, secondary, accent)
- Typography settings (fonts, sizes)
- Layout options (sidebar, content width)
- Dark mode support
- Custom CSS injection

### Feedback System

Collect user feedback:

- "Was this page helpful?" widget
- Custom feedback forms
- Response analytics
- Export functionality

### LLM Integration

AI-friendly documentation:

- Auto-generated `llms.txt`
- Complete `llms-full.txt`
- Configurable content inclusion

### SEO Optimization

- Custom page titles and descriptions
- OpenGraph meta tags
- Structured data
- Clean URLs

### Admin Panel

Complete management interface:

- Dashboard with statistics
- User management (admin-only)
- Site settings (theme, typography, layout, branding, advanced)
- Activity logs with filtering and export
- Media manager with folder organization
- Two-factor authentication support

### Security

- Two-factor authentication (TOTP)
- Cloudflare Turnstile CAPTCHA support
- Rate limiting on sensitive endpoints
- Content Security Policy headers
- Activity logging with IP detection

## Tech Stack

### Frontend
- Vue 3 with Composition API
- TypeScript
- Inertia.js
- Tailwind CSS
- TipTap Editor
- Vite

### Backend
- Laravel 12
- PHP 8.4+
- MySQL/PostgreSQL/SQLite
- Redis (optional)

## Installation

```bash
git clone https://github.com/crony-io/docs-starter-kit.git
cd docs-starter-kit
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm run dev
```
